# Beanerys
Software Group Project-Semester 5

# Project Definition : 
Website for Local Restaurants (Beanerys’)

# Description : 
This website is for the ease of the customers so they do not need to wait for hours for their snacks and beverages. They can pre-order their food via this website so they don’t have to wait in line for their order and also they can pay for their food online.

# Purpose and Scope :
Purpose of this project/website is to provide convenience to client for them to give order of food for their customers with managing their time and resources.
